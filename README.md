## Clone repo
```bash 
git clone -b android-12.1 https://github.com/Andy312432/android_device_samsung_sm8650_twrp device/samsung/e1q
```

## To build 
```bash
export ALLOW_MISSING_DEPENDENCIES=true
. build/envsetup.sh
lunch twrp_e1q-eng
mka recoveryimage
```
